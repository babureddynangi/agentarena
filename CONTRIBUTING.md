# Contribution Guidelines

Thank you for your interest in contributing to the Agent Arena! Please follow the guidelines below:

## How to Contribute

1. **Fork the repository**: Start by forking the Agent Arena repository to your own GitHub account.
2. **Create a new branch**: Make a new branch for your work (preferably named according to the feature you are working on).
3. **Make your changes**: Implement your feature or fix in your branch.
4. **Commit your changes**: Use clear and descriptive commit messages to explain your changes.
5. **Push to your fork**: Push your changes to your forked repository.
6. **Create a pull request**: Submit a pull request (PR) to the main repository's main branch.

## Code of Conduct

Please adhere to the [Code of Conduct](link_to_code_of_conduct) in all interactions related to this project.

## Reporting Issues

If you encounter any issues, please report them on the [Issues page](<link_to_issues_page>). Provide as much detail as possible.

## Thank You!

Thank you for contributing to the Agent Arena! Your contributions help improve the project and are greatly appreciated.
