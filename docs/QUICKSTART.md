# Quickstart Guide for AgentArena

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/babureddynangi/agentarena.git
   ```
2. Navigate into the project directory:
   ```bash
   cd agentarena
   ```
3. Install the necessary dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Running Tests
To run the tests, execute the following command:
```bash
pytest
```

## Project Structure
- **docs/**: Contains the documentation files.
- **src/**: The source code of the project.
- **tests/**: The test cases for the project.
- **requirements.txt**: Lists the package dependencies.

## Next Steps
- Explore the source code in the `src/` directory and familiarize yourself with the components of the project.
- Check the `tests/` directory for sample test cases and how to write your own.
- Contribute to the project by fixing issues or adding new features!
